Construye tu web a con variables CSS a tiempo real

DWR proporciona variedad de colores, tipografías y más para usarlas donde quieras. Puedes modificar los estilos según tus preferencias.

Usa cualquiera de las variables globales para escribir nuevos estilos. Estas usan la sintaxis var(--global-variableName).

:root{
--global-bg: #ffffff;
--global-text: #222;
--global-accent: #4aa3a6;
--global-muted: #6b6b6b;
--global-radius: 8px;
--global-gap: 1rem;
--global-container-width: 1100px;
--global-shadow: 0 6px 18px rgba(17,17,17,0.06);
--global-transition: 0.18s ease;
--global-font: "Inter", "Helvetica Neue", Arial, sans-serif;
}

Estiliza los componentes

Añade cualquiera de las clases predefinidas a los componentes para modificar su vista, como los botones en el ejemplo de abajo. Existe variedad de clases disponibles. Combínalas para tener estilos más avanzados.

</section>
