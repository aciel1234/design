Construye tu web con variables CSS en tiempo real

DWR proporciona colores, tipografías y más para usarlas donde quieras.

Usa cualquiera de las variables globales con la sintaxis var(--global-variableName).

:root {
--global-brown1: #362828;
--global-radius: 8px;
--global-gap: 1rem;
--global-shadow: 0 6px 18px rgba(17,17,17,0.06);
--global-transition: 0.3s ease;
--global-font4: 'Times New Roman', Times, serif;
}
